# TrendX Logo Assets

## Files Included

### Primary Logos
- `trendx-logo.svg` - Main logo with gradient colors (512x512px)
- `trendx-logo-dark.svg` - Dark mode variant with white/light colors
- `trendx-app-icon.svg` - Simplified app icon version (1024x1024px)

## Usage Guidelines

### Primary Logo (`trendx-logo.svg`)
- Use on light backgrounds
- Minimum size: 32px for mobile visibility
- Scalable vector format for all screen sizes
- Features gradient from electric blue (#00D4FF) to purple (#8B5CF6)

### Dark Mode Logo (`trendx-logo-dark.svg`)
- Use on dark backgrounds
- White and light gray colors for contrast
- Same dimensions and styling as primary logo

### App Icon (`trendx-app-icon.svg`)
- Optimized for mobile app icons
- Bold, simplified design works at small sizes
- Square format with rounded corners
- Dark gradient background with prominent X symbol

## Color Specifications

### Primary Colors
- Electric Blue: `#00D4FF`
- Purple: `#8B5CF6`
- Deep Blue: `#0F172A`
- Dark Purple: `#1E1B4B`

### Light Mode Colors
- White: `#FFFFFF`
- Light Gray: `#F8FAFC`

## Technical Notes

- All logos are vector-based SVG format
- Scalable from 16px to 1024px without quality loss
- Transparent backgrounds for versatility
- Optimized for web and mobile applications
- Compatible with modern browsers and design tools

## Implementation

```html
<!-- Primary logo -->
<img src="assets/logo/trendx-logo.svg" alt="TrendX" width="200" height="200">

<!-- Dark mode logo -->
<img src="assets/logo/trendx-logo-dark.svg" alt="TrendX" width="200" height="200">

<!-- App icon -->
<img src="assets/logo/trendx-app-icon.svg" alt="TrendX" width="64" height="64">
```

## Export Formats Needed

To generate PNG versions for different platforms:

1. **iOS App Icon**: Export `trendx-app-icon.svg` as 1024x1024px PNG
2. **Android App Icon**: Export `trendx-app-icon.svg` as 512x512px PNG
3. **Favicon**: Export primary logo as 32x32px and 16x16px PNG
4. **Social Media**: Export primary logo as 1200x630px PNG for sharing
